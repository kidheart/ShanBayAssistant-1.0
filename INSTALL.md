# 安装指南

## 快速开始

### 步骤 1: 准备图标文件（可选但推荐）

在安装扩展之前，建议先添加图标文件：

1. 打开 `icons/` 文件夹
2. 添加以下三个 PNG 图标文件：
   - `icon16.png` (16x16 像素)
   - `icon48.png` (48x48 像素)
   - `icon128.png` (128x128 像素)

**如果没有图标文件：**
- 扩展仍然可以正常使用
- 只是会显示默认图标
- 可以在后续添加图标后刷新扩展

### 步骤 2: 加载扩展到 Chrome

1. **打开 Chrome 扩展管理页面**
   ```
   在地址栏输入：chrome://extensions/
   ```
   或通过菜单：`更多工具` → `扩展程序`

2. **启用开发者模式**
   - 在页面右上角，打开"开发者模式"开关

3. **加载扩展**
   - 点击"加载已解压的扩展程序"按钮
   - 选择项目文件夹（`CURSORenglish`）
   - 点击"选择文件夹"

4. **验证安装**
   - 扩展应该出现在扩展列表中
   - 浏览器工具栏会出现扩展图标（如果有图标文件）

### 步骤 3: 开始使用

1. **打开扇贝单词网站**
   ```
   https://www.shanbay.com
   ```

2. **进入学习页面**
   - 登录并进入背单词页面

3. **按下热键**
   - Windows/Linux: `Ctrl + Shift + I`
   - Mac: `Cmd + Shift + I`

4. **查看图片**
   - 右侧会弹出图片窗口
   - 显示当前单词的相关图片

## 常见问题

### Q: 提示"无法加载扩展"？
A: 检查以下几点：
- 确保 `manifest.json` 文件存在且格式正确
- 检查所有引用的文件是否存在
- 查看 Chrome 扩展页面的错误信息

### Q: 热键不工作？
A: 尝试以下方法：
1. 刷新扇贝单词页面
2. 检查是否在扇贝单词域名下
3. 确认扩展已启用
4. 尝试手动点击扩展图标（如果支持）

### Q: 无法识别单词？
A: 这可能是由于：
- 扇贝单词页面结构发生变化
- 需要使用开发者工具检查实际的 HTML 结构
- 可能需要更新 `content.js` 中的选择器

### Q: 图片无法显示？
A: 可能原因：
- 网络连接问题
- Unsplash 服务暂时不可用
- 可以点击"在 Google 图片中搜索"链接手动查看

## 卸载扩展

1. 打开 `chrome://extensions/`
2. 找到"扇贝单词图片助手"
3. 点击"移除"按钮
4. 确认删除

## 更新扩展

当有新版本时：

1. 下载新版本代码
2. 在扩展管理页面点击"重新加载"按钮（刷新图标）
3. 或先移除再重新加载

## 开发者模式调试

如果需要调试扩展：

1. 在扩展管理页面点击"检查视图" → "service worker"
2. 打开扇贝单词页面，按 F12 打开开发者工具
3. 在 Console 中查看错误信息
4. 修改代码后，在扩展管理页面点击"重新加载"
