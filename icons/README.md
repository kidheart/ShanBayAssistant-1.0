# 图标文件说明

## 需要的图标尺寸

- `icon16.png` - 16x16 像素（工具栏图标）
- `icon48.png` - 48x48 像素（扩展管理页面）
- `icon128.png` - 128x128 像素（Chrome 网上应用店）

## 如何创建图标

### 方法一：使用在线工具

1. 访问 [Favicon Generator](https://favicon.io/favicon-generator/) 或类似工具
2. 创建或上传你的图标设计
3. 下载不同尺寸的 PNG 文件
4. 重命名为对应的文件名并放在此文件夹

### 方法二：使用设计软件

使用 Photoshop、GIMP、Figma 等工具创建图标：
- 设计一个简单的搜索图标或单词图标
- 导出为 PNG 格式，分别保存为 16x16、48x48、128x128 三种尺寸

### 方法三：使用临时占位图标

如果暂时没有图标，可以使用以下方法：

1. 创建一个简单的文本图标（在临时文件夹）
2. 或者暂时使用 Chrome 默认图标（扩展仍可正常使用）

## 图标设计建议

- 使用搜索图标（🔍）或单词图标（📚）
- 主色调：紫色/蓝色系（与界面风格一致）
- 简洁明了，在小尺寸下也能清晰识别
